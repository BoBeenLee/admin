## 1. 개요

| 항목 | 내용 |
|------|------|
| **페이지명** | 대시보드 캘린더 (홈) |
| **파일** | `home.html` |
| **역할** | 지점장이 한 눈에 오늘의 현황을 파악하고 빠르게 액션을 취할 수 있는 통합 대시보드 |

| **진입 경로** | 앱 최초 진입 시 기본 페이지, 사이드 네비게이션 "대시보드 캘린더 (홈)" |

## 5. 인터랙션 명세

| 인터랙션 | 트리거 | 동작 |
|----------|--------|------|
| 드로어 열기 | 햄버거 클릭 | 오버레이 페이드인 + 드로어 슬라이드인 |
| 드로어 닫기 | X 버튼 / 오버레이 클릭 | 드로어 슬라이드아웃 + 오버레이 페이드아웃 |
| 모달 열기 | `openScheduleModal()` | 오버레이 + 모달 스케일 애니메이션 |
| 모달 닫기 | `closeScheduleModal()` | 역순 애니메이션 → hidden |
| 프로필 열기 | 이름 클릭 `openProfileModal(name)` | 모달 오픈 + 데이터 바인딩 |
| 전화번호 복사 | 전화번호 클릭 `copyPhone()` | 클립보드 복사 + 하단 토스트 1.5초 |
| 날짜 탭 전환 | 탭 클릭 / 스와이프 | 해당 날짜의 리더 목록 렌더링 |
| 필터 적용 | 언어/시간/상태 버튼 | 리더 목록 실시간 필터링 |
| 메모 편집 | 메모 영역 클릭 → 타이핑 → 포커스아웃 | `_homeSaveMemo()` 자동 저장 |

## 7. 현재 제약사항 (Known Limitations)

1. **데이터**: 클라이언트 사이드 Mock 데이터 (40명 리더 랜덤 생성), 백엔드 미연동
2. **캘린더**: 월 전환 미구현 (4월 고정), 주간 뷰 미구현
3. **공지사항**: 하드코딩 2개 항목, CRUD 미구현
4. **리마인드**: 실제 SMS/메시지 발송 미연동
5. **프로필**: 3명만 상세 데이터 보유 (김지윤, 박선민, 성경희), 나머지는 김지윤 fallback
