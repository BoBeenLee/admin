## A. 대시보드 (home.html)

### A-1. 출근 후 오늘 현황 파악

> **사용자:** 지점장
> **목적:** 출근 직후 오늘 지점의 전반적인 상태를 빠르게 확인

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 앱 접속 (home.html 자동 진입) | 대시보드 로드, 4개 위젯 + 캘린더 표시 |
| 2 | 공지사항 위젯 확인 | 중요 공지 자동 스크롤로 최신 공지 확인 |
| 3 | 리더 현황 위젯 확인 | 오늘 배정된 리더 수, 출석예정/완료/불참 카운트 확인 |
| 4 | 당일 첫방문 스터디 위젯 확인 | 오늘 첫 방문하는 신규 멤버 이름과 반 확인 |
| 5 | 내일 리마인드 위젯 확인 | 내일 신규 멤버 발송 대상 인원 확인 |

**완료 조건:** 지점장이 오늘의 리더 현황, 신규 멤버, 주요 공지를 1분 내에 파악

---

### A-2. 리더 현황 상세 확인 (7일)

> **사용자:** 지점장
> **목적:** 이번 주 리더 출석 상태를 날짜별로 확인하고 불참 리더 대응

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 리더 현황 위젯의 "크게 보기" 클릭 | 리더 현황 보드 모달 오픈 |
| 2 | 오늘(기본) 탭에서 리더 목록 확인 | 상태별 요약 바 + 리더 행 목록 표시 |
| 3 | "불참" 상태 필터 클릭 | 불참 리더만 필터링되어 표시 |
| 4 | 불참 리더의 전화번호 클릭 | 전화번호 클립보드 복사 + 토스트 알림 |
| 5 | 내일(+1일) 탭 클릭 | 내일 배정 리더의 출석 예정 현황 표시 |
| 6 | "영어" 언어 필터 클릭 | 영어 팀 리더만 필터링 |
| 7 | "11시" 시간 필터 클릭 | 영어 11시 팀 리더만 표시 |
| 8 | 특정 리더 이름 클릭 | 해당 리더의 프로필 모달 오픈 |

**완료 조건:** 불참 리더를 파악하고 연락처를 확보하여 대타 배정 준비

---

### A-3. 신규 멤버 리마인드 전송

> **사용자:** 매니저
> **목적:** 내일 첫 방문 예정인 신규 멤버에게 리마인드 메시지 전송

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | "내일 신규 멤버 리마인드" 위젯 확인 | 발송 대상 아바타 스택 + 인원 수 표시 |
| 2 | "전체 퀵 전송" 버튼 클릭 | 대상 멤버 전원에게 리마인드 메시지 발송 |

**완료 조건:** 내일 신규 멤버 전원에게 리마인드 발송 완료

---

### A-4. 캘린더에서 월간 일정 확인

> **사용자:** 지점장
> **목적:** 이번 달 신규 등록, 상담, 행사 일정을 한눈에 확인

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 캘린더 영역 스크롤 | 월간 캘린더 표시 (4월) |
| 2 | 파란색 이벤트 확인 | 신규 등록 멤버: 시간 + 이름 + 전화번호 |
| 3 | 초록색 이벤트 확인 | 상담 일정 확인 |
| 4 | 주황색 이벤트 확인 | 행사 일정 (예: 리더 간담회 회식) 확인 |
| 5 | 신규 등록 이벤트의 전화번호 클릭 | 해당 멤버 전화번호 복사 |

**완료 조건:** 월간 주요 일정을 색상별로 구분하여 파악

---

### A-5. 특정 멤버 프로필 확인

> **사용자:** 지점장
> **목적:** 리더 현황 보드에서 특정 리더의 상세 프로필 확인

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 리더 현황 모달에서 리더 이름 클릭 | 프로필 모달 오픈 |
| 2 | 통계 카드 확인 | 최초등록일, 멤버십(잔여), 시작일, 만료일, 최근출석률 확인 |
| 3 | 상담 코멘트 확인 | 관리자가 남긴 최근 상담 메모 확인 |
| 4 | 히스토리 타임라인 스크롤 | 멤버십 변경, 레벨업, 지점 이동 등 전체 이력 확인 |
| 5 | "개별 문자" 버튼 클릭 | 해당 멤버에게 개별 SMS 발송 |

**완료 조건:** 멤버의 등록 이력, 현재 상태, 상담 내용을 종합 파악

## B. 리더 팀 (leader.html)

### B-1. 특정 팀 리더 출석부 확인

> **사용자:** 지점장
> **목적:** 특정 언어/시간대 팀의 리더 출석 현황 확인

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 사이드 메뉴에서 "리더 팀" 클릭 | leader.html 이동, "전체" 탭 활성 |
| 2 | 영어 "화 8시" 팀 탭 클릭 | 해당 팀 리더만 필터링 표시 |
| 3 | 출석 테이블에서 리더별 출석/불참/대타 수치 확인 | 초록/빨강/주황 숫자로 한눈에 상태 파악 |
| 4 | 최근 출석률 블록 호버 | 각 회차별 날짜, 상태, 참여 멤버 툴팁 표시 |
| 5 | 출석률이 낮은 리더 확인 | 빨간색 퍼센트로 표시된 리더 식별 |

**완료 조건:** 특정 팀의 리더별 출석 상태와 문제 리더 식별

---

### B-2. 불참 리더 대응 및 대타 관리

> **사용자:** 지점장
> **목적:** 불참 예정 리더를 파악하고 대타를 배정

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 리더 현황 보드 (좌측) 확인 | 오늘 배정 리더 상태별 요약 |
| 2 | "불참" 상태 필터 클릭 | 불참 리더만 필터링 |
| 3 | 리더 불참 일정 보드 (우측) 확인 | 불참 리더별 불참 날짜 목록 확인 |
| 4 | 불참 리더의 전화번호 클릭 | 클립보드 복사 → 전화로 대타 요청 |
| 5 | 대타 확정 후 불참 날짜 체크박스 클릭 | 해당 날짜 취소선 + 초록 체크 전환 |
| 6 | 모든 불참 날짜 해결 | 해당 리더 행 페이드아웃 자동 제거, 카운트 갱신 |

**완료 조건:** 모든 불참 리더에 대해 대타 배정 또는 해결 처리 완료

---

### B-3. 리더 검색 및 정렬

> **사용자:** 매니저
> **목적:** 특정 리더를 빠르게 찾거나, 출석률 기준으로 리더 목록 정렬

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 검색창에 "박선민" 입력 | 실시간으로 "박선민" 리더 행만 표시 |
| 2 | 검색 클리어 (X 버튼) | 전체 리더 목록 복원 |
| 3 | 정렬을 "출석률 낮은순"으로 변경 | 출석률이 가장 낮은 리더부터 정렬 |
| 4 | 리딩LV 필터를 "LV1"로 변경 | LV1 리더만 필터링 |
| 5 | 정렬을 "금일 미출석순"으로 변경 | 오늘 아직 출석하지 않은 리더 우선 표시 |

**완료 조건:** 원하는 조건의 리더를 빠르게 검색/필터링/정렬

---

### B-4. 리더 당일 출석 체크

> **사용자:** 매니저
> **목적:** 리더가 세션에 도착했을 때 당일 출석 체크

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 해당 리더 행의 "금일 출석" 버튼 클릭 | 회색 → 초록 체크 전환 |
| 2 | 리더 현황 보드에서 상태 확인 | 해당 리더 상태가 "출석 완료"로 반영 |

**완료 조건:** 리더의 당일 출석이 정확히 기록됨

---

### B-5. 리더 메모 수정

> **사용자:** 지점장
> **목적:** 특정 리더에 대한 관리 메모를 업데이트

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 리더 행의 메모 셀 클릭 | 인라인 편집 모드 활성화 (커서 진입) |
| 2 | 기존 메모 수정 또는 새 메모 작성 | 텍스트 입력 가능 |
| 3 | 메모 셀 외부 클릭 (포커스아웃) | 자동 저장, 연필 아이콘 사라짐 |

**완료 조건:** 메모가 즉시 저장되어 다음 조회 시에도 유지

---

### B-6. 리더에게 일괄 문자 전송

> **사용자:** 지점장
> **목적:** 특정 팀의 전체 리더에게 공지 문자 전송

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 해당 팀 탭 선택 | 팀 리더 목록 표시 |
| 2 | 상단 "일괄 문자" 버튼 클릭 | SMS 모달 오픈 |
| 3 | 메시지 내용 작성 후 전송 | 해당 팀 전체 리더에게 SMS 발송 |

**대안 흐름 (선택 발송):**

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 원하는 리더들의 체크박스 선택 | 벌크 액션 바 표시 "N명 선택됨" |
| 2 | 벌크 액션 바의 "문자" 버튼 클릭 | 선택된 리더에게만 SMS 발송 |

**완료 조건:** 대상 리더에게 문자 발송 완료

### B-7. 7일간 리더 현황 스와이프 확인 (모바일)

## C. 멤버 팀 (member.html)

### C-1. 팀별 멤버 출석부 확인

> **사용자:** 매니저
> **목적:** 특정 팀의 레벨별 멤버 출석 현황 확인

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 사이드 메뉴에서 "멤버 팀" 클릭 | member.html 이동, 영어 월수 11시 탭 기본 활성 |
| 2 | "일어" 영역의 "화목 11시" 팀 탭 클릭 | 일어 화목 11시 팀으로 전환, 레벨Nav + 테이블 재렌더링 |
| 3 | 레벨 바로가기에서 "LV 3" 클릭 | LV 3 섹션으로 스무스 스크롤 |
| 4 | LV 3 테이블에서 멤버 출석 블록 확인 | 주당 출석 블록 시각화로 상태 확인 |

**완료 조건:** 특정 팀의 레벨별 멤버 출석 현황을 체계적으로 파악

---

### C-2. 주의 알림 (CHECK) 대응

> **사용자:** 지점장
> **목적:** 긴급 조치가 필요한 멤버를 확인하고 대응

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | CHECK 섹션 확인 | 알림 유형별 (첫방문미체크, 만료D-7, 출석률부진 등) 멤버 목록 표시 |
| 2 | "첫방문 미체크" 멤버 이름 클릭 | 해당 멤버 행으로 자동 스크롤 + 하이라이트 |
| 3 | 멤버 행에서 출석 체크 수행 | 금일 출석 버튼 → 초록 전환 |
| 4 | CHECK 섹션으로 복귀하여 "해결" 클릭 | 해당 알림 항목 제거, 카운트 -1 |
| 5 | "만료 D-7" 멤버의 전화번호 클릭 | 복사 → 연장 등록 안내 전화 |
| 6 | 상담 후 "해결" 클릭 | 알림 제거 |

**완료 조건:** 모든 CHECK 알림에 대해 조치 완료 (알림 카운트 0)

---

### C-3. 멤버십 만료 임박 멤버 관리

> **사용자:** 지점장
> **목적:** 곧 만료되는 멤버를 파악하고 연장 등록 유도

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 정렬을 "만료 임박순"으로 변경 | 만료일이 가장 가까운 멤버부터 정렬 |
| 2 | 멤버십 필터를 "H+"로 변경 | H+ 등급 (52회, 소규모) 멤버만 필터링 |
| 3 | 만료 임박 멤버의 이름 클릭 | 프로필 모달 오픈 |
| 4 | 프로필에서 잔여 세션 + 만료일 확인 | "잔여 2/52회", "만료 26.04.09" 등 확인 |
| 5 | 상담 코멘트 확인 | 이전 상담 기록으로 맥락 파악 |
| 6 | "개별 문자" 버튼 클릭 | 연장 등록 안내 SMS 발송 |

**완료 조건:** 만료 임박 멤버 파악 및 연장 안내 발송

---

### C-4. 멤버십 등급별 현황 파악

> **사용자:** 지점장
> **목적:** 특정 멤버십 등급의 멤버 분포 및 출석 현황 파악

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 멤버십 필터를 "VVIP"로 변경 | VVIP 멤버만 필터링 |
| 2 | 정렬을 "출석률 낮은순"으로 변경 | 관리가 필요한 VVIP 멤버 우선 표시 |
| 3 | 출석률이 낮은 VVIP 멤버 확인 | 빨간색 퍼센트로 표시된 멤버 식별 |
| 4 | 해당 멤버 프로필 확인 | 잔여 세션이 많은데 출석률이 낮은 패턴 확인 |
| 5 | 멤버십 필터를 "B"로 변경 | B등급 (24회) 소규모 멤버 확인 |
| 6 | 정렬을 "잔여 적은순"으로 변경 | 곧 세션이 소진되는 멤버 파악 |

**완료 조건:** 등급별 멤버 현황을 파악하고 관리 우선순위 결정

### C-5. 멤버 당일 출석 일괄 체크

> **사용자:** 매니저
> **목적:** 오전 세션 시작 시 참석한 멤버들의 출석을 빠르게 체크

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 해당 팀 탭 선택 (예: 영어 월수 11시) | 팀 멤버 표시 |
| 2 | 정렬을 "금일 미출석순"으로 변경 | 아직 체크하지 않은 멤버 우선 표시 |
| 3 | 참석한 멤버의 "금일 출석" 버튼 클릭 | 회색 → 초록 체크 전환 |
| 4 | 반복하여 전체 참석 멤버 체크 | 모든 참석 멤버 초록 체크 표시 |
| 5 | 정렬을 "금일 출석 완료순"으로 변경 | 체크 완료된 멤버 상단, 미출석 멤버 하단 확인 |

**완료 조건:** 참석 멤버 전원의 당일 출석 체크 완료

### C-6. 멤버 팀 이동 (일괄)

|------|------|----------|

| 2 | "이동" 버튼 클릭 | 팀 선택 드롭다운 표시 |
| 3 | 이동할 팀 선택 후 확인 | 선택 멤버들이 새 팀으로 이동 |

**완료 조건:** 선택 멤버들의 팀 배정 변경 완료

### C-7. 팀 배지 클릭으로 팀 간 이동

| 2 | 전환된 팀에서 해당 멤버 확인 | 다른 팀에서의 출석률/레벨/상태 확인 |

### C-8. 멤버 상세 프로필 및 히스토리 확인

> **사용자:** 지점장
> **목적:** 상담 전 멤버의 전체 이력을 사전 파악

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | 멤버 이름 클릭 | 프로필 모달 오픈 |
| 2 | 통계 카드 확인 | 최초등록일, 멤버십&횟수(잔여/총), 시작&만료일, 누적출석률 |
| 3 | 상담 코멘트 확인 | 이전 관리자의 상담 기록 + 주의사항 |
| 4 | 히스토리 타임라인 스크롤 | 정기점검 → 상담기록 → 최초등록 순서로 전체 이력 확인 |
| 5 | 전화번호 클릭 | 클립보드 복사 → 상담 전화 |
| 6 | "정보 수정" 클릭 | 회원 정보 수정 페이지 이동 |

**완료 조건:** 멤버의 전체 맥락(등록, 멤버십, 출석, 상담) 파악 후 상담 준비 완료

### D-2. 신규 멤버 첫 방문일 관리 (풀 사이클)

> **사용자:** 지점장 + 매니저
> **목적:** 신규 멤버의 등록부터 첫 방문 체크까지 전체 프로세스

| 단계 | 담당 | 행동 | 기대 결과 |
|------|------|------|----------|
| 1 | 지점장 | home.html 캘린더에서 내일 신규 등록(파란 이벤트) 확인 | 신규 멤버 이름 + 연락처 파악 |
| 2 | 지점장 | "내일 리마인드" 위젯의 "전체 퀵 전송" 클릭 | 리마인드 메시지 발송 |
| 3 | 매니저 | (다음 날) home.html "당일 첫방문 스터디" 위젯 확인 | 오늘 첫 방문 멤버 리스트 확인 |
| 4 | 매니저 | 햄버거 메뉴 → "멤버 팀" 이동 | member.html 이동 |
| 5 | 매니저 | CHECK 섹션에서 "첫방문 미체크" 알림 확인 | 해당 멤버의 알림 표시 |
| 6 | 매니저 | 멤버 이름 클릭 → 해당 행 스크롤 | 멤버 행 하이라이트 |
| 7 | 매니저 | 금일 출석 버튼 클릭 | 출석 체크 완료 |
| 8 | 매니저 | CHECK 섹션 → "해결" 클릭 | 첫방문 알림 제거 |

**완료 조건:** 신규 멤버의 리마인드 → 첫 방문 → 출석 체크 → 알림 해결 전체 사이클 완료

### D-3. 출석률 부진 멤버 상담 프로세스

> **사용자:** 지점장
> **목적:** 출석률이 낮은 멤버를 식별하고 상담 후 관리

| 단계 | 행동 | 기대 결과 |
|------|------|----------|
| 1 | member.html에서 정렬 "출석률 낮은순" 선택 | 출석률 최하위 멤버 상단 표시 |
| 2 | CHECK 섹션에서 "출석률 부진" 알림 확인 | 출석률 20% 이하 멤버 목록 |
| 3 | 해당 멤버 이름 클릭 → 프로필 모달 | 상세 프로필 확인 |
| 4 | 히스토리에서 최근 상담 기록 확인 | 이전 상담 내용으로 맥락 파악 |
| 5 | 전화번호 복사 → 전화 상담 | 출석 독려 상담 진행 |
| 6 | 상담 완료 후 멤버 행의 메모 셀 수정 | "상담 완료. 다음 주부터 출석률 개선 약속" 메모 저장 |
| 7 | CHECK 섹션 → "해결" 클릭 | 출석률 부진 알림 제거 |

**완료 조건:** 부진 멤버 식별 → 상담 → 메모 기록 → 알림 해결
