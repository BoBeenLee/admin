## 한 줄 요약

> **60명 리더의 팀별/날짜별 출석을 한 화면에서 추적하고, 불참 → 대타 워크플로우를 클릭 몇 번에 마무리하는 출석부.**

| 항목 | 내용 |
|------|------|
| 주요 Actor | **지점장** (대응) · **매니저** (체크) |
| 진입 경로 | 사이드 메뉴 → "리더 팀" |
| 핵심 가치 | "팀별 출석 한눈에" → "불참→대타 워크플로우" → "인라인 메모 즉시 저장" |

## 핵심 유저 스토리 (5)

### 🟥 P0 · B-1 특정 팀 리더 출석부 확인

> **"영어 화 8시 팀의 리더가 누가 안 나오고 있는지 보고 싶다."**

| 항목 | 내용 |
|------|------|
| Actor | 지점장 |
| 트리거 | 사이드 메뉴 → "리더 팀" → 팀 탭 클릭 |
| 완료 조건 | 출석률 낮은 리더 식별 |

```mermaid
flowchart LR
    A([사이드 메뉴]) --> B[리더 팀 진입<br/>전체 탭 활성]
    B --> C[영어 화 8시 탭 클릭]
    C --> D[해당 팀 필터링]
    D --> E[테이블에서 출석/불참/대타 수치 확인]
    E --> F[출석 블록 호버<br/>회차별 툴팁]
    F --> G{출석률 낮은 리더?}
    G -->|Yes| H[빨간 % 식별 → 추적]
    G -->|No| I([정상])
```

### 🟧 P1 · B-2 불참 리더 대응 및 대타 관리 ⭐

> **"불참 통보 받은 리더의 일정을 확인하고, 대타를 배정해서 빨리 정리하고 싶다."** — 페이지 핵심 워크플로우

| 항목 | 내용 |
|------|------|
| Actor | 지점장 |
| 트리거 | 좌측 보드 "불참" 필터 / 우측 불참 일정 보드 |
| 완료 조건 | 모든 불참 날짜에 대타 배정 또는 해결 처리 |

**🎨 baoyu-diagram SVG (다크 테마):**

![B-2 불참 → 대타 배정 워크플로우](./diagrams/leader-b2-substitute-workflow.svg)

**📐 Mermaid (라이트 테마, 인라인):**

```mermaid
sequenceDiagram
    actor 지점장
    participant 현황보드 as 좌측 현황 보드
    participant 불참보드 as 우측 불참 일정
    participant Phone as 클립보드
    participant 리더 as 불참 리더

    지점장->>현황보드: 상태 필터 = 불참
    현황보드-->>지점장: 불참 리더 목록 표시
    지점장->>불참보드: 불참 일정 카드 확인
    불참보드-->>지점장: 날짜별 체크박스 노출
    지점장->>Phone: 전화번호 클릭 (복사)
    Phone-->>지점장: 토스트 1.5s
    지점장->>리더: 통화 → 대타 요청
    리더-->>지점장: 대타 확정
    지점장->>불참보드: 해당 날짜 체크박스 클릭
    불참보드-->>지점장: 취소선 + 초록 체크
    Note over 불참보드: 모든 날짜 해결 시<br/>0.4s 페이드아웃
```

### 🟥 P0 · B-4 리더 당일 출석 체크

> **"리더가 도착하면 한 클릭으로 출석 기록한다."**

```mermaid
flowchart LR

    C --> D[회색 → 초록 토글]
    D --> E[좌측 현황 보드 동기화]
    E --> F([기록 완료])
```

### 🟦 P2 · B-6 리더 일괄 문자 전송

> **"이번 주 토요일 휴강 공지를 영어 토 11시 팀 리더 전체에게 보낸다."**

| 항목 | 내용 |
|------|------|
| Actor | 지점장 |
| 트리거 | 팀 탭 선택 후 "일괄 문자" 또는 체크박스 → 벌크 액션 바 |
| 완료 조건 | 대상 리더에게 SMS 발송 |

```mermaid
flowchart TD
    Start([공지 필요]) --> Choose{대상}
    Choose -->|팀 전체| TeamTab[팀 탭 선택]
    Choose -->|선택 발송| Checks[체크박스 다중 선택]

    TeamTab --> BtnAll[일괄 문자 버튼]
    Checks --> Bulk[벌크 액션 바 노출<br/>N명 선택됨]
    Bulk --> BtnSel[문자 버튼]

    BtnAll --> Modal[SMS 모달]
    BtnSel --> Modal
    Modal --> Send([전송 완료])
```

## 페이지 동작 핵심 트리거 요약

| UI 액션 | 결과 | JS 핸들러 |
|---------|------|----------|
| 팀 탭 클릭 | 해당 팀 리더만 필터링 | `data-team` 속성 기반 |
| 날짜 탭 클릭 / 스와이프(50px) | 7일 보드 날짜 전환 | 좌우 스와이프 핸들러 |
| 상태/언어/시간 필터 | 즉시 리더 행 재렌더링 | 멀티 필터 적용 |
| 메모 셀 포커스아웃 | 자동 저장 | contenteditable + blur |
| 출석 블록 호버 | 고정 위치 툴팁 | mouseenter/leave |
| 행 호버 | translateY(-1px) + 그림자 | `.member-row` |
| 불참 체크박스 | 취소선 + 초록 / 모두 해결 시 페이드아웃 | 0.4s transition |
