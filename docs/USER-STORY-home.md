> **"출근하자마자 오늘 하루를 1분 안에 그림으로 잡고 싶다."**

| 항목 | 내용 |
|------|------|
| Actor | 지점장 |
| 트리거 | 앱 진입 (자동) |
| 완료 조건 | 1분 내 리더·신규·공지 파악 완료 |

```mermaid
flowchart LR
    A([앱 진입]) --> B[공지사항 위젯<br/>중요 공지 자동 스크롤]
    B --> C[리더 현황 위젯<br/>출석예정/완료/불참 카운트]
    C --> D[당일 첫방문 위젯<br/>오늘 신규 멤버]
    D --> E[내일 리마인드 위젯<br/>발송 대상 인원]
    E --> F([현황 파악 완료])
```

### 🟧 P1 · A-2 리더 현황 상세 확인 (7일)

> **"이번 주 어떤 리더가 불참인지, 누구에게 대타 요청할지 파악하고 싶다."**

| 항목 | 내용 |
|------|------|
| Actor | 지점장 |
| 트리거 | 리더 현황 위젯 → "크게 보기" |
| 완료 조건 | 불참 리더 식별 + 연락처 확보 |

**🎨 baoyu-diagram SVG (다크 테마):**

![A-2 리더 현황 상세 확인 워크플로우](./diagrams/home-a2-leader-board-flow.svg)

**📐 Mermaid (라이트 테마, 인라인):**

```mermaid
flowchart TD
    Start([지점장 출근]) --> Widget[리더 현황 위젯 확인]
    Widget --> Big{불참자 발견?}
    Big -->|Yes| Modal[크게 보기 → 모달 오픈]
    Big -->|No| End1([다른 위젯으로])
    Modal --> Today[오늘 탭 기본 활성]
    Today --> FilterStatus[상태 필터: 불참]
    FilterStatus --> FilterLang[언어 필터: 영어/일어]
    FilterLang --> FilterTime[시간 필터: 11시/8시]
    FilterTime --> Phone[전화번호 클릭 → 클립보드 복사]
    Phone --> Action[대타 배정 통화]
    Action --> Complete([대응 완료])
```

> 📖 상세 단계: [USER-SCENARIOS.md#a-2](./USER-SCENARIOS.md#a-2-리더-현황-상세-확인-7일)

### 🟥 P0 · A-5 특정 멤버/리더 프로필 확인

> **"리더 한 명의 등록 이력·상담·출석률을 종합해서 보고 싶다."**

| 항목 | 내용 |
|------|------|
| Actor | 지점장 |
| 트리거 | 리더 현황 모달 또는 캘린더 이벤트의 이름 클릭 |
| 완료 조건 | 멤버 맥락(등록·멤버십·출석·상담) 파악 |

```mermaid
flowchart TD
    A[리더 현황 모달] --> B[리더 이름 클릭]
    B --> C[프로필 모달 오픈]
    C --> D[통계 카드 5개<br/>최초등록 · 멤버십 · 시작 · 만료 · 출석률]
    D --> E[상담 코멘트<br/>최근 메모 · 주의사항]
    E --> F[히스토리 타임라인<br/>등록 → 변경 → 이동]
    F --> G{후속 액션}
    G -->|연락| H[전화번호 복사]
    G -->|개별 메시지| I[개별 문자 버튼]
    G -->|닫기| J([파악 완료])
```
