| 항목 | 내용 |

| **프로젝트명** | 하남 지점 어드민 (Hanam Branch Admin) |

## 3. 비즈니스 도메인 (5개)

### 3-1. 멤버 관리 (Member Management) `복잡도: 높음`

어학원에 등록된 학생(멤버)을 관리합니다.

**핵심 엔티티:** 멤버, 멤버십 등급, 리딩 레벨, 세션 횟수

**비즈니스 규칙:**
- 멤버십 등급별 총 세션 수: VVIP(1040회), VIP(520회), A+(260회), H+(120회), B(60회)
- 멤버는 언어별 리딩 레벨 LV1~LV5로 분류
- 멤버십 만료 D-7에 알림 발생
- 신규 멤버의 첫 방문은 반드시 체크 확인 필요

**관련 파일:** [member.html](../member.html)

---

### 3-2. 리더 관리 (Leader Management) `복잡도: 보통`

스터디 세션을 이끄는 리더(튜터/진행자)를 관리합니다.

**핵심 엔티티:** 리더, 팀 배정, 출석 상태, 대타

**비즈니스 규칙:**
- 리더 출석 상태 5가지: 출석예정, 출석완료, 대타예정, 대타출석완료, 불참
- 리더 현황 보드는 7일 롤링 뷰로 언어/시간 필터 지원
- 불참 리더는 대타 배정 추적 필요

**관련 파일:** [leader.html](../leader.html)

---

### 3-3. 출석 추적 (Attendance Tracking) `복잡도: 보통`

멤버와 리더 모두의 세션별 출석을 추적합니다.

**핵심 엔티티:** 출석 기록, 출석 블록, 출석률

**비즈니스 규칙:**
- 출석 상태: 출석예정 / 출석완료 / 대타예정 / 대타출석완료 / 불참
- 최근 출석률은 8블록 시각화로 표시
- 누적 출석률은 멤버 프로필에서 추적

**관련 파일:** [member.html](../member.html), [leader.html](../leader.html)

---

### 3-4. 팀 스케줄링 (Team Scheduling) `복잡도: 보통`

언어(영어/일어)와 요일+시간대로 스터디 팀을 구성합니다.

**핵심 엔티티:** 팀, 타임슬롯, 언어 그룹

**비즈니스 규칙:**
- 팀 = 언어 + 요일 + 시간 (예: 영어 월수 11시, 일어 화목 8시)
- 멤버와 리더가 특정 팀 슬롯에 배정
- 일괄 작업(이동, 문자, 삭제)을 선택된 멤버/리더에 적용 가능

**관련 파일:** [member.html](../member.html), [leader.html](../leader.html)

---

### 3-5. 대시보드 & 커뮤니케이션 `복잡도: 보통`

메인 대시보드에서 캘린더, 공지사항, 리더 현황, 신규 멤버 리마인드, SMS 발송을 제공합니다.

**핵심 엔티티:** 공지사항, 캘린더 이벤트, SMS 메시지, 리마인더

**비즈니스 규칙:**
- 캘린더 색상 코드: 파란색(신규 등록), 초록색(상담), 주황색(행사)
- 내일 신규 멤버 리마인드 퀵 전송 기능
- 선택된 멤버/리더에게 일괄 SMS 발송 가능

**관련 파일:** [home.html](../home.html)

## 5. 도메인 간 상호작용

```
                    ┌─────────────────────┐
                    │  대시보드 & 커뮤니케이션  │
                    └──────┬──────┬────────┘
                           │      │
              멤버 현황 집계 │      │ 리더 현황 집계
                           ▼      ▼
              ┌────────────┐      ┌────────────┐
              │  멤버 관리   │      │  리더 관리   │
              └──────┬──┬──┘      └──┬──┬──────┘
                     │  │            │  │
        팀 배정 ─────┘  │            │  └───── 팀 배정
                        │            │
                        ▼            ▼
               ┌────────────────────────────┐
               │       출석 추적              │
               └────────────────────────────┘
                        ▲            ▲
                        │            │
              ┌─────────┴────────────┴─────────┐
              │         팀 스케줄링              │
              └────────────────────────────────┘
```

- **멤버 관리** ↔ **팀 스케줄링**: 멤버가 언어+시간 기반 팀에 배정됨
- **리더 관리** ↔ **팀 스케줄링**: 리더가 팀에 배정되어 세션 진행
- **멤버 관리** → **출석 추적**: 세션별 멤버 출석률 시각화 및 일일 체크인
- **리더 관리** → **출석 추적**: 대타 처리 포함 리더 출석 7일 현황 뷰
- **대시보드** → **멤버 관리**: 첫 방문 멤버, 신규 멤버 리마인드 표시
- **대시보드** → **리더 관리**: 리더 현황 오버뷰 및 확장 보드

## 6. 가이드 투어 (추천 학습 순서)

새로 합류한 팀원은 아래 순서로 코드를 탐색하면 전체 시스템을 빠르게 이해할 수 있습니다.

### Step 1: 대시보드부터 시작
**파일:** [home.html](../home.html)

대시보드는 시스템의 전체 모습을 한눈에 보여줍니다. 위젯 구조(공지, 리더 현황, 첫 방문 스터디, 리마인드)와 캘린더를 먼저 파악하세요.

### Step 2: 리더 현황 보드 이해
**파일:** [home.html](../home.html) (하단 스크립트)

`openScheduleModal()`, `buildSummary()`, `renderRow()` 함수들이 리더 데이터를 어떻게 생성하고 렌더링하는지 확인하세요. 출석 상태 5가지와 7일 롤링 뷰 로직이 핵심입니다.

### Step 3: 리더 팀 출석부
**파일:** [leader.html](../leader.html)

팀 탭 필터링, 리더 출석 테이블, 불참 일정 보드를 살펴보세요. 8블록 출석률 시각화와 대타 시스템에 주목하세요.

### Step 4: 멤버 팀 출석부
**파일:** [member.html](../member.html)

멤버 관리의 핵심 페이지입니다. 리딩 레벨별 섹션 그룹핑, CHECK 알림 섹션(첫 방문 미체크, 만료 D-7), 멤버십 등급 시스템을 파악하세요.

### Step 5: 프로필 모달
**파일:** 세 파일 모두에 존재

`openProfileModal()` 함수와 모달 구조를 확인하세요. 등록일, 멤버십, 잔여 세션, 출석률, 상담 코멘트, 히스토리 타임라인이 포함됩니다.

### Step 6: 공통 패턴 파악
세 페이지에 걸쳐 반복되는 패턴들:
- **사이드 네비게이션 드로어**: 동일한 구조와 토글 로직
- **검색 & 필터**: 이름/연락처 검색, 정렬, 필터링
- **일괄 작업**: 체크박스 선택 → 이동/문자/삭제
- **클릭 투 에디트**: 메모 필드의 인라인 수정
- **연락처 복사**: `copyPhone()` 함수

아래 영역은 복잡도가 높아 수정 시 주의가 필요합니다.

## 8. 용어 사전

| 한글 | 영문 | 설명 |
|------|------|------|
| 멤버 | Member | 어학원에 등록한 학생 |
| 리더 | Leader | 스터디 세션을 이끄는 튜터/진행자 |
| 팀 | Team | 언어 + 요일 + 시간으로 구성된 스터디 그룹 |
| 리딩 레벨 | Reading Level | LV1~LV5의 언어 능력 등급 |
| 멤버십 | Membership | VVIP / VIP / A+ / H+ / B 등급 |
| 출석예정 | Expected | 해당 세션에 출석 예정 |
| 출석완료 | Completed | 출석 확인 완료 |
| 대타 | Substitute | 다른 리더가 대신 진행 |
| 불참 | Absent | 세션 불참 |
| 첫 방문 | First Visit | 신규 멤버의 첫 스터디 참여 |
| CHECK | Alert | 주의가 필요한 멤버 알림 |
