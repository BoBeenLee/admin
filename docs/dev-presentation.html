06 — Comparison

leader.html vs member.html 비교

구조는 유사하지만 핵심 차이점이 존재합니다

항목 leader.html (리더) member.html (멤버)
대상 인원 ~60명 ~1,500명 (팀당 LV당 30명)
팀 구조 요일+시간 (영 월 11시) 일정+시간 (영어 월수 11시)
레벨 그룹핑 ✕ 단일 테이블 ✓ LV0~LV4 섹션 분리
레벨 바로가기 ✕ ✓ Sticky Quick-Nav
현황 보드 ✓ 7일 스와이프 + 불참 보드 ✕
CHECK 알림 ✕ ✓ 5종 알림
멤버십 표시 ✕ ✓ 등급 + 잔여 + 만료
출석 블록 수 고정 8블록 가변 8 또는 16블록
데이터 생성 Math.random() Seeded PRNG (Mulberry32)
정렬 옵션 8종 11종 (잔여/만료 추가)
🔄 공유 컴포넌트 (재사용 가능)
글래스 헤더 사이드 드로어 프로필 모달 전화번호 복사 + 토스트 출석 블록 시각화 출석 블록 툴팁 메모 인라인 편집 금일출석 토글 벌크 액션 바 SMS 모달 팀 배지 LV 배지
